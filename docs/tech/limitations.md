# Known Limitations

- Current setup is _not_ pixel perfect, as the intention is mimic the original screen.
- There is a bug on the game _Top Ranking Tennis_ (only one detected). This game uses pseudo digitalized voices and due nature of the emulator the sound is buggous. No solution found at the time of writing this lines.
- No copyrighted games are included. Use only ROMs you legally own.
- The emulator targets original Game Boy DMG software, not Game Boy Color or Game Boy Advance.
- Full-speed operation depends on aggressive optimization and may trade away some emulation accuracy.
- Some games may not behave exactly like original hardware.
- Repeated flashing will eventually wear flash memory.
- Overclocking may reduce long-term hardware lifespan.
- Incorrect wiring or voltage mistakes can damage the Pico or attached peripherals.
